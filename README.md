##认识OBS##
*OBS全称Open Broadcaster Software，它比各大平台自己的直播软件功能更强大
*下载地址：https://obsproject.com/
*下载后直接打开，根据步骤安装即可，在此不多赘述

*安装完成后，关闭欢迎向导，就能看到OBS的主界面
